# golang-simple-server-sample

A simple HTTP server sample written in Go.
I built it to test [golang-stats-api-handler][stats],
demonstrating graceful shutdown and structured access logging.
It is published as Docker images, both [Docker Hub][docker-hub]
and [GitHub Container Registry][ghcr].
It is also my playground for the Docker image release flow.
Do not use it in production.

[stats]: https://github.com/fukata/golang-stats-api-handler
[docker-hub]: https://hub.docker.com/r/itchyny/golang-simple-server-sample
[ghcr]: https://github.com/users/itchyny/packages/container/package/golang-simple-server-sample
